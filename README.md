# labo
Créer une classe Souris qui permet de construire des souris, de les faire vieillir et évoluer, avec des comportements spécifiques pour les clones.
Attributs :
poids (double, en grammes)
couleur (string)
age (unsigned int, en mois)
esperanc_vie (unsigned int, durée de vie en mois)
clonee (booléen, indique si la souris est clonée)
